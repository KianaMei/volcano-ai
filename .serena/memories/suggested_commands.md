# 建议命令

- **后端运行**: cd chat-service; mvn spring-boot:run
- **前端开发**: cd chat-widget; npm run dev
- **构建**: cd chat-service; mvn package; cd chat-widget; npm run build
- **测试**: (未指定, 假设 mvn test for backend)
- **Git**: git status, git add ., git commit -m \"msg\", git push
